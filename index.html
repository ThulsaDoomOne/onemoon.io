ONEMOON BUYING GUIDE

If you have BNB or ETH in metamask, send it to Binance.com, Crypto.com or Kucoin.com and trade it for $ONE instead. This is faster and cheaper than using the Harmony bridge.

Set up metamask to use the Harmony network. Click settings > Networks > Add Network then copy the settings here:

  • Network Name: Harmony Mainnet
  • RPC URL: https://api.harmony.one/
  • Chain ID: 1666600000
  • Symbol: ONE
  • Block Explorer URL: https://explorer.harmony.one/#/

Copy your metamask address, paste it into explorer.harmony.one and get the Harmony version (it will start with 'one1' instead of '0x') use this address to withdraw ONE from your exchange. Once your wallet is funded with ONE coins you are ready for the next step.

Go to https://one.mochiswap.io/#/swap, set your slippage to 12%, then select ‘ONEMOON’ from the token drop down menu -- if you don’t see it, you can search for it using the contract address: 0xCB35e4945c7F463c5CCBE3BF9f0389ab9321248F

Select how many ONEMOON you will like to purchase, then click ‘approve’ and ‘swap’ to initiate the exchange.

Final step is to add ONEMOON to your metamask wallet for tracking by clicking ‘add token’ > ‘custom token’ and pasting in the contract address here: 0xCB35e4945c7F463c5CCBE3BF9f0389ab9321248F
